Propriedade position no CSS e Como Posicionar Elementos

Introdução

No CSS, a propriedade position é usada para definir como os elementos serão posicionados na página. Ela oferece controle sobre o posicionamento, permitindo criar layouts mais dinâmicos e precisos.

Valores da Propriedade position

  • Static
    • Definição: É o valor padrão. O elemento é posicionado de acordo com o fluxo da página
    • Exemplo: 
                              
div {
    position: static
}
  • Relative
    • Definição: O elemento é posicionado em relação à sua posição original no fluxo da página. Utiliza as propriedades top, right, bottom e left para ajustar o deslocamento a partir da posição inicial.
    • Exemplo: 
                              
.box {
    position: relative;
    top: 20px; /* Move 20px para baixo da posição original */
    left: 30px; /* Move 30px para a direita */
}
  • Absolute
    • Definição: O elemento é fixado em relação ao seu container pai mais próximo que tenha a propriedade position diferente de static.
    • Exemplo: 
                              
.parent {
    position: relative;
}
.child {
    positino: absolute;
    top:10px; /* 10px do topo do container pai */
}
