HTML

html (linguagem de mercado de hipertexto) é o bloco de construção mais básico da web. Defin e o significado e a assinatura de conteúdo web.

TAG ou Marcação

o html usa marcaçao para anotar texto, imagem e outros conteudos para exibiçao em um navegador web

um elemento html é separado de outro texto em um documento por tags. a tags consistem no nome do elemento entre < e >
